# Initialize the QA Generator

## Supported Formats

**Semantic QA Gen** supports the following file formats:

- **PDF**: Portable Document Format
- **DOCX**: Microsoft Word Open XML Document
- **MD**: Markdown Document
- **TXT**: Plain Text File
